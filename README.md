# 📿 تسبيح - مسبحة رقمية عصرية وفاخرة

<!-- ![Tasbih App Banner](assets/images/banner.png) -->

"تسبيح" هو تطبيق فلاتر (Flutter) عصري وفاخر، تم تصميمه ليقدم تجربة روحانية وسلسة للذكر. يجمع التطبيق بين الطمأنينة الروحية وأحدث تقنيات تصميم تطبيقات الهاتف المحمول، معتمداً على فلسفة تصميم "Liquid Glass" (الزجاج السائل).

---

## ✨ المميزات الرئيسية

- **💎 واجهة Liquid Glass**: جمالية بصرية تعتمد على تأثير الزجاج المصنفر مع تدرجات لونية ناعمة وعناصر تفاعلية جذابة.
- **📊 إحصائيات ذكية**: تتبع تقدمك الروحاني بإحصائيات مفصلة (اليومية، الإجمالي، والمتوسط). اكتشف "أفضل يوم" لك وحافظ على استمراريتك.
- **🎨 محرك سمات ديناميكي**: واجهة قابلة للتخصيص بالكامل من خلال أربع فئات ألوان أساسية:
  - الخلفية وأشرطة التطبيق (Background & AppBars)
  - الخطوط والمحتوى (Fonts & Content)
  - الأيقونات واللمسات الإضافية (Icons & Accents)
  - البطاقات والعناصر التفاعلية (Interactive Cards)
- **🔄 إدارة بيانات متقدمة**: نظام هجين لإدارة البيانات باستخدام **SQLite** لتتبع السجل التاريخي، و **SharedPreferences** للأداء الفوري السريع.
- **🎬 حركات (Animations) فاخرة**: انتقالات سلسة وحركات دقيقة مدعومة بحزم `flutter_animate` و `animations`.
- **🌍 دعم كامل للغة العربية (RTL)**: تجربة مستخدم مثالية مصممة خصيصاً للمستخدم العربي مع خط **Cairo** الأنيق.
- **🔔 استجابة لمسية (Haptic Feedback)**: تفاعل حسي يجعل تجربة الذكر أكثر واقعية وتأثيراً.
- **🔋 وضع الاستيقاظ (Wakelock)**: الحفاظ على الشاشة نشطة أثناء جلسات الذكر الطويلة لضمان التركيز وعدم المقاطعة.

## 🛠️ التقنيات المستخدمة

- **إطار العمل**: [Flutter](https://flutter.dev)
- **إدارة الحالة**: [Riverpod](https://riverpod.dev) (حديث وتفاعلي)
- **قواعد البيانات**: [sqflite](https://pub.dev/packages/sqflite) (تخزين محلي بجودة عالية)
- **الحركات**: `flutter_animate` و `animations`
- **الرسوم البيانية**: `fl_chart` لعرض البيانات بصرياً
- **الخطوط**: خط Cairo (من Google Fonts)

## 🚀 البداية السريعة

### المتطلبات الأساسية

- Flutter SDK (أحدث إصدار)
- Android Studio أو VS Code
- جهاز حقيقي أو محاكي (Emulator)

### التثبيت

1. **تحميل المشروع**:
   ```bash
   git clone https://github.com/yourusername/tasbih.git
   ```

2. **الانتقال إلى مجلد المشروع**:
   ```bash
   cd tasbih
   ```

3. **تحميل المكتبات**:
   ```bash
   flutter pub get
   ```

4. **تشغيل التطبيق**:
   ```bash
   flutter run
   ```
